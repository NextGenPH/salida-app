# SALIDA

SALIDA is a Netflix-inspired streaming platform built with Next.js, featuring movie and TV show browsing, watchlists, and trailers.

## Features
- Browse Trending, Popular, and Top Rated movies.
- Search for movies.
- Watchlist management (saved locally to your browser).
- Multi-server player support (VidSrc, VidLink).
- Trailer integration.
- Mobile-first, responsive design.
- PWA support (installable as an app).

## Getting Started

1. **Clone the repository.**
2. **Install dependencies:** `npm install`
3. **Configure environment:** Copy `.env.example` to `.env` and add your `NEXT_PUBLIC_TMDB_API_KEY`.
4. **Run development server:** `npm run dev`

## Deployment
This project is optimized for deployment on Vercel.
